# Airline-Management-System

## PROJECT OVERVIEW
This Project is a desktop-based Airline Management System developed using Python, MySQL and Tkinter GUI

The system allows airline to be tracked to the customer via the departure and arrival times. It also integrates the admin button to make add, remove or make changes whenever required.

## Technologies Used
- Python
- MySQL
- Tkinter (GUI)
- MySQL Connector

## FEATURES
- Admin Login System
- Add / Remove / Delete and make changes on Flight Records
- User Panel
- Track Flight Details
- Database Integration with MySQL
